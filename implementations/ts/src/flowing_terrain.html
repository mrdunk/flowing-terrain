This project is intended to document a simple algorithm for creating 3 dimensional terrain maps and their likely watercourses.
The map you see is randomly generated. For a new map, regenerate one of the sources under the Geography tab.
Cursor keys and mouse to move around.
An explanation of the algorithm can be found here: https://github.com/mrdunk/flowing-terrain
This demonstration was made with the excellent Babylon.js web rendering library.
